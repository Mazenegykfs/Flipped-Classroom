Using Question Marks in Technical Writing

1. Objectives

  • Correctly use a question mark to end a direct question (an interrogative sentence).
  • Differentiate between a direct question and an indirect question.
  • Avoid using question marks at the end of statements or commands.
  • Understand that direct questions are used for inquiry, not for stating facts in technical reports.

2. Values & Life Skills

Values

  • Clarity: Using correct punctuation makes your writing unambiguous.
  • Precision: Choosing the right end mark conveys the exact intended tone.
  • Professionalism: Proper grammar and punctuation are hallmarks of professional communication.

Life Skills

  • Effective Communication: Asking clear questions and reporting information accurately.
  • Attention to Detail: Recognizing subtle but important grammatical rules.
  • Critical Thinking: Analyzing sentence structure to determine its function (statement vs. inquiry).

3. Review

A question mark (?) is an end punctuation mark that is used to indicate a direct question. It signals to the reader that a response is being sought.

4. Warm-up

How does the punctuation change the function of these sentences?

1. The calibration is complete.

2. Is the calibration complete?

Sentence 1 is a declarative statement that provides information. Sentence 2 is an interrogative sentence that asks for information. The punctuation fundamentally changes the sentence's purpose.

5. Key Vocabulary

  • Direct Question (Interrogative Sentence): A sentence that asks for information directly and ends with a question mark (?).
    Example: Have you checked the logs?
  • Indirect Question: A statement that reports or describes a question. It ends with a period (.).
    Example: The supervisor asked if I had checked the logs.
  • Declarative Sentence: A sentence that makes a statement or declares something. It ends with a period (.).
    Example: I have checked the logs.

6. Structure: Two Key Question Mark Rules

Rule 1: Use for Direct Questions

A question mark is used at the end of an interrogative sentence, which is a sentence that asks a direct question.

Example: What were the results of the stress test?

Example: Have you submitted the final report?

Rule 2: Do NOT Use for Indirect Questions

An indirect question is a statement that reports a question. It does not ask the question directly to the reader, so it ends with a period.

Correct: The manager asked if the report was finished.

Incorrect: The manager asked if the report was finished?
