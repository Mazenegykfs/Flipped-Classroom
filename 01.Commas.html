Using Commas in Technical Writing

1. Objectives

  • Use commas correctly to separate items in a series.
  • Use commas with coordinating conjunctions to join independent clauses.
  • Use commas to set off introductory elements from the main clause.
  • Use commas to set off non-essential clauses and phrases.

2. Review

A comma (,) is a punctuation mark that indicates a brief pause in a sentence. It helps to separate ideas, phrases, and clauses, making sentences easier to read and understand.

3. Warm-up

How does the comma change the meaning of this sentence?

1. The project requires experienced engineers and managers.

2. The project requires experienced engineers, and managers.

Sentence 1 implies that both the engineers and managers must be experienced. Sentence 2 could imply that the engineers must be experienced, but the managers might not be. Commas clarify meaning!

4. Presentation: Four Key Comma Rules

Rule 1: Commas in a Series

Use commas to separate three or more words, phrases, or clauses in a list.

Example: The design requires steel, copper, and aluminum.

Note: The final comma before 'and' is called the Oxford or serial comma. In technical writing, it is highly recommended to prevent ambiguity.

Rule 2: Comma Before a Coordinating Conjunction

Use a comma before a coordinating conjunction (for, and, nor, but, or, yet, so) when it joins two independent clauses.

Example: The test results were positive, but the team decided to run a second verification.

Rule 3: Comma After an Introductory Element

Use a comma to separate an introductory word, phrase, or clause from the main clause of the sentence.

Example: After analyzing the data, the engineers identified the primary fault.

Rule 4: Commas with Non-Essential Elements

Use a pair of commas to set off clauses, phrases, and words that are not essential to the meaning of the sentence.

Example: The main turbine, which was recently serviced, is operating at peak efficiency.
